Өгөгдлийн урьдчилсан судалгаа (Data exploration)¶

Энэхүү лабораторийн ажлаар өгөгдөл судлахад (data exploration) зориулсан Python кодын жишээг агуулна. Data exploration гэдэг нь өгөгдлийн онцлог шинж чанарыг илүү сайн ойлгохын тулд хийгдэх өгөгдлийн урьдчилсан судалгааг хэлнэ. Өгөгдлийг урьдчилан судалснаар:

  • Хэрэглэгчдэд тохирох урьдчилсан боловсруулалт, өгөгдөлд шинжилгээ хийх аргыг сонгоход нь туслах.
  • Өгөгдөл дэх хэв маягийг танихад хүмүүсийн чадварыг ашиглах.

Доорх зааварчилгааг алхам алхмаар анхааралтай уншина уу.

Тойм статистикүүд (Summary statistics)¶

Тойм статистикүүд нь өгөгдлийн багцын товч тойм эсвэл хураангуйг өгдөг тодорхойлогч хэмжүүрүүдийг хэлнэ. Жишээ нь дундаж (mean), стандарт хазайлт (standard deviation) гэх мэт. Энэхүү зааварт бид Setosa, Versicolour, Virginika гэсэн гурван төрлийн цахилдаг цэцэг тус бүрээс 50 ширхэг, нийт 150 ширхэг цэцгийн талаарх мэдээллийг агуулсан Цахилдаг цэцгийн түүврийн өгөгдлийг ашиглана. Цэцэг бүр дараах таван шинж чанараар тодорхойлогддог.

Аттрибутын мэдээлэл:

  • sepal урт /сантиметр/

  • sepal өргөн /сантиметр/

  • petal урт /сантиметр/

  • petal өргөн /сантиметр/

  • Ангилал (Setosa, Versicolour, Virginica)

Энэхүү ажлаар дараах зүйлсийг сурна.

  • CSV өгөдлийн файлыг уншиж, Pandas DataFrame обьектод хадгалах
  • DataFrame обьектод хадгалагдсан өгөгдөл дээр янз бүрийн тойм статистикийг тооцоолох

1. Дараах код нь CSV файлыг уншиж, DataFrame объектод хадгалахад ашиглагддаг. Дараа нь өгөгдлийн эхний таван мөрийг харуулах болно.

from pandas.api.types import is_numeric_dtype

for col in data.columns:
    if is_numeric_dtype(data[col]):
        print('%s:' % (col))
        print('\t Mean = %.2f' % data[col].mean())
        print('\t Standard deviation = %.2f' % data[col].std())
        print('\t Minimum = %.2f' % data[col].min())
        print('\t Maximum = %.2f' % data[col].max())

Өгөгдлийн дүрслэл¶

I. Iris өгөгдлийн 3 төрлийн цэцэг бүрийн 4 атрибутын дундаж утгыг barplot графикаар дүрслэн харуулна. График зурахад дараах аргуудын аль нэгийг ашиглаж болно:

a) matplotlib сангийн pyplot модулийг ашиглах.

b) эсвэл seaborn сангийн barplot() функцийг ашиглах.

Seaborn бол matplotlib дээр суурилсан өгөгдлийн дүрслэлийн Python сан юм. Энэ нь мэдээлэл сайтай, статистик график зурах өндөр түвшний интерфейсээр хангадаг.

II. scatter plot-н өргөжүүлсэн хувилбар болох дараах графикийг мөн дээрх өгөгдөл дээр дүрсэлж харна уу.

III. Heat map нь өгөгдлийн утгуудыг матриц хэлбэрээр өнгөөр кодлон дүрсэлдэг график дүрслэлийн арга юм. Энэ нь хоёр хэмжээст матриц дахь тоон утгуудад өнгө оноох замаар атрибутуудын хоорондын хамаарлыг ойлгомжтой байдлаар илэрхийлдэг.

Дээр өгөгдсөн Iris dataset-ийн атрибутуудын хоорондын корреляцийн матрицыг тооцоолж, seaborn сангийн heatmap() функцийг ашиглан дүрслэн харуул.

Ижил болон ялгаатай байдал¶

IV. Eucledian distance: Евклидийн зай бол хамгийн түгээмэл хэрэглэгддэг зайн хэмжүүр юм. - Евклидийн зайг олох функц тодорхойлно уу - Дараах 2 обьектын хоорондох зайг олно уу.

x = [0,3,4,5]
y = [7,6,3,-1]

V. Manhattan distance:

p1 нь (x1, y1), p2 нь (x2, y2) үед
Манхэттэн зай = |x1 – x2| + |y1 – y2|

- Манхэттэний зайг олох функц тодорхойлно уу.
- Дараах 2 обьектын хоорондох зайг олно уу.

x = [10,20,10]
y = [10,20,20]

VI.. Minkowski distance: Минковскийн зай нь Евклидийн зай болон Манхэттэний зайн ерөнхий хэлбэр. - r = 1 Манхэттэний зай - r = 2 Евклидийн зай - r = ∞ Чебышевын зай 

- Минковскийн зайг олох функц бич.
- Дараах 2 обьектын хоорондох зайг олно уу (r = 3).


x = [0,3,4,5]
y = [7,6,3,-1]

VII. Cosine similarity Үүнийг түгээмэл ашиглах болсон нэг шалтгаан нь сийрэг векторуудыг үнэлэхэд маш үр дүнтэй байдаг.

- Cosine similarity олох функц бич
- Дараах 2 обьектын косинусын ижил төстэй байдлыг олно уу.

x = [3, 45, 7, 2]
y = [2, 54, 13, 15]

VIII. Jaccard Similarity: Жаккардын ижил төстэй байдал нь хязгаарлагдмал түүврийн олонлогуудын ижил төстэй байдлыг хэмждэг. 

- Jaccard Similarity олох функцийг тодорхойл  / set.intersection(), set.union() /
- Дараах 2 обьектын Жаккардын ижил төстэй байдлыг олно уу


x = [0,1,2,5,6]
y = [0,2,3,5,7,9]
