Cluster Analysis¶

Кластерын шинжилгээ нь оролтын өгөгдлийг хоорондоо нягт холбоотой тохиолдлуудын бүлгүүдэд хуваахыг эрмэлздэг бөгөөд ингэснээр нэг кластерт хамаарах тохиолдлууд нь бусад кластерт хамаарах тохиолдлуудаас илүү төстэй байх болно.

15.1 K-means Clustering¶

k-means алгоритм нь кластер бүрийг түүний харгалзах кластерын төвөөр (centroid) илэрхийлдэг. Алгоритм нь дараах хоёр алхмыг давталттайгаар хэрэгжүүлэх замаар оролтын өгөгдлийг k салангид кластерт хуваах болно.

  1. Цэг бүрийг хамгийн ойрын центройд руу оноож k кластер үүсгэх.
  2. Кластер бүрийн центройдыг дахин тооцоолох.

Энэ хэсэгт бид киноны үнэлгээний өгөгдлийн багцын жишээн дээр k-means кластер хийж гүйцэтгэнэ. Эхлээд өгөгдлийн багцыг дараах байдлаар үүсгэе.

import pandas as pd

ratings = [['john',5,5,2,1],['mary',4,5,3,2],['bob',4,4,4,3],['lisa',2,2,4,5],['lee',1,2,3,4],['harry',2,1,5,5]]
titles = ['user','Jaws','Star Wars','Exorcist','Omen']

Жишээ өгөгдлийн багцад эхний 3 хэрэглэгч тулаант кинонд (Jaws болон Star Wars) дуртай байсан бол сүүлийн 3 хэрэглэгчид аймшгийн кино (Exorcist болон Omen) таалагдсан байна. Бидний зорилго бол ижил төстэй киноны сонголттой хэрэглэгчдийн бүлгийг тодорхойлохын тулд хэрэглэгчдэд k-means кластерийг ашиглах явдал юм.

Доорх жишээ нь киноны үнэлгээний өгөгдөлд k-means кластерийг (k=2) хэрхэн ашиглахыг харуулна. Кластер хийх алгоритмыг хэрэглэхийн өмнө эхлээд "user" баганыг арилгах ёстой. Хэрэглэгч бүрийн кластерын хуваарилалтыг dataframe объект хэлбэрээр харуулъя. Кластерийн анхны төвийг тодорхойлоход 'k-means++' болон 'random' сонголтууд байх ба бид энэ тохиолдолд 'k-means++' аргыг ашигласан.

In [ ]:
from sklearn import cluster

data = movies.drop('user',axis=1)
X = data.to_numpy()

k_means = cluster.KMeans(n_clusters=2, init = 'k-means++', max_iter=50, random_state=42)
k_means.fit(X) 
labels = k_means.labels_
pd.DataFrame(labels, index=movies.user, columns=['Cluster ID'])

k-means кластерийн алгоритм нь эхний гурван хэрэглэгчийг нэг кластерт, сүүлийн гурван хэрэглэгчийг хоёр дахь кластерт хуваарилсан байна. Өөрөөр хэлбэл үр дүн нь бидний хүлээлттэй нийцэж байна. Мөн хоёр кластер бүрийн центройдыг харуулж болно.

In [ ]:
centroids = k_means.cluster_centers_
pd.DataFrame(centroids,columns=data.columns)

Кластер 0 нь аймшгийн киноны хувьд илүү өндөр үнэлгээтэй, харин 1-р кластер тулаант киноны хувьд илүү өндөр үнэлгээтэй байгааг ажиглаарай. Кластерын центройд нь бусад хэрэглэгчдэд тэдний кластерыг тодорхойлох даалгаварт хэрэглэгдэж болно.

Өгөгдөл дэх кластеруудын оновчтой тоог тодорхойлохын тулд k-means алгоритмыг 1-ээс 6 хүртэлх кластерын янз бүрийн тоотой хэрэглэж, тэдгээрийн харгалзах алдааны квадратуудын нийлбэрийг (SSE-sum of squared errors) доорх жишээнд үзүүлсэн шиг тооцоолж болно. Кластерын тоог үнэлэхийн тулд SSE-ийг кластерын тоотой харьцуулсан график дахь "elbow"-г ашиглаж болно. "elbow" нь гол өөрчлөлтийг олж авахад тустай байдаг ба графикийг хүний гартай зүйрлэвэл тохой буюу нугарч байгаа цэгийг "elbow" гэж үзнэ.

import matplotlib.pyplot as plt
%matplotlib inline

numClusters = [1,2,3,4,5,6]
SSE = []
for k in numClusters:
    k_means = cluster.KMeans(n_clusters=k, n_init=1)
    k_means.fit(data)
    SSE.append(k_means.inertia_)  # SSE

plt.plot(numClusters, SSE)
plt.xlabel('Number of Clusters')
plt.ylabel('SSE')
plt.scatter(2,SSE[1], s = 200, c = 'red', marker='*')

Elbow аргыг ашигласнаар кластерийн оновчтой тоо 2 гэдгийг баталгаажууллаа.

15.2 Hierarchical Clustering¶

Энэ хэсэгт өмнөх лабораторид ашиглагдаж байсан сээр нуруутан амьтдын өгөгдлийн багцад шаталсан кластерыг ашиглана. Тодруулбал, Python scipy сангаар хангагддаг шаталсан кластерийн 3 алгоритмыг ашигласан үр дүнг харуулна: (1) single link (MIN), (2) complete link (MAX), (3) group average. Уг сангаас хангадаг бусад шаталсан кластерын алгоритмууд нь центройдод суурилсан болон Ward-ийн аргыг агуулдаг.

15.3 Density-Based Clustering¶

Нягтад суурилсан кластер нь бага нягтралтай бүсээр тусгаарлагдсан өндөр нягтралтай бүсийг тодорхойлдог. DBSCAN бол хамгийн алдартай нягтад суурилсан алгоритмуудын нэг юм. DBSCAN-д өгөгдлийн цэгүүдийг 3 төрөлд ангилагдана.

  1. Гол цэгүүд (core points)
  2. Хилийн цэгүүд (border point)
  3. Шуугианы цэгүүд (noisy point)

2 параметрийг тодорхойлно: радиус (eps) болон уг радиус дотор байх цэгийн минимум тоо (min_samples) Энэ аргад зориулан дараах шуугиантай, 2 хэмжээст өгөгдлийн багцыг ашиглана. Дараах код нь өгөгдлийг ачаалж, тархалтын графикийг харуулна.

In [ ]:
import pandas as pd

data = pd.read_csv('chameleon.data', delimiter=' ', names=['x','y'])
data.plot.scatter(x='x',y='y')

eps=15.5, min_samples=5 гэсэн тохируулгатайгаар DBScan кластерийн алгоримтыг өгөгдөл дээр хэрэгжүүлнэ. Кластеруудад 0-8 хооронд ID буюу дугаар оноох ба шуугианы цэгүүдийг (noise points) ID нь -1 тэнцүү кластерт хуваарилна.

15.4 Spectral Clustering¶

k-means кластерын алгоритмын гол хязгаарлалтуудын нэг нь бөмбөрцөг хэлбэртэй кластеруудыг эрэлхийлэх хандлага юм. Тиймээс дурын хэлбэртэй кластер бүхий өгөгдлийн багцад ашиглах эсвэл кластерын төвүүд (centroids) хоорондоо давхцах үед энэ нь ажиллахгүй. Ижил төстэй байдлын графын (similarity graph) шинж чанарыг ашиглан spectral clustering нь энэ хязгаарлалтыг даван туулж чадна. Үүнийг харуулахын тулд дараах хоёр хэмжээст өгөгдлийн багцыг авч үзье.

In [ ]:

data1 = pd.read_csv('2d_data.txt', delimiter=' ', names=['x','y'])

Дээрх графикууд нь k-means кластерийн гүйцэтгэл муу байгааг харуулж байна. Харин одоо өгөгдлийн багцад spectral clustering ашиглацгаая. Доорх жишээнд Gaussian radial basis функцийг ижил төстэй байдлын (affinity) хэмжүүр болгон ашиглана. Өгөгдсөн өгөгдлийн багцад тохирох кластеруудыг авахын тулд хэрэглэгч цөмийн параметрийн (gamma) утгыг тааруулах шаардлагатай.

from sklearn import cluster
import pandas as pd

spectral = cluster.SpectralClustering(n_clusters=2,random_state=1,affinity='rbf',gamma=5000)
spectral.fit(data1)
labels1 = pd.DataFrame(spectral.labels_,columns=['Cluster ID'])
result1 = pd.concat((data1,labels1), axis=1)

spectral2 = cluster.SpectralClustering(n_clusters=2,random_state=1,affinity='rbf',gamma=5000)
spectral2.fit(data2)

result2 = pd.concat((data2,labels2), axis=1)

fig, (ax1,ax2) = plt.subplots(nrows=1, ncols=2, figsize=(12,5))

Даалгавар¶

Худалдааны төвийн хэрэглэгчийн шошгогүй (label) өгөгдөл дээр k-means кластерийн алгоритмыг ашиглан худалдааны төвөөс тодорхойлсон жилийн зарцуулалт, зарцуулалтын оноонд үндэслэн үйлчлүүлэгчдийн бүлгийг үүсгэх болно. Зарцуулалтын оноо нь 1-ээс 100 хүртэл ба үүнийг худалдааны төвөөс үйлчлүүлэгчийн зан байдал, зарцуулалтын шинж чанарт үндэслэн оноодог.

Ашиглах өгөгдөл: customers.csv

  1. Өгөгдлийг Dataframe-д авч, эхний 5 бичлэгийг хадгал
  2. Өгөгдлийг scatter plot дээр дараах байдлаар дүрсэлж харна уу.

  1. Зөвхөн жилийн орлого, зарлагын онооны багануудыг ашиглах тул эдгээр баганы утгуудыг авч, өгөгдлөө бэлдэнэ үү.
  2. Кластерийн тоог 1-11 хүртэл авч, "elbow" аргыг ашиглан кластерийн оновчтой тоог тодорхойл. Кластерийн анхны төвийг олохдоо k-means++ аргыг, max_iter дефаулт утгыг тус тус ашиглана.
  3. Кластерийн тооны оновчтой утгыг ашиглан K-Means Clustering тооцоолох.
  1. Өгөгдлийн багц дахь бичлэг бүр аль кластерийн гишүүн болсныг дараах байдлаар хэвлэж хар.

  2. (3) даалгаварт бэлдсэн өгөгдлөөр нягтад суурилсан кластер үүсгэ. eps = 10, min_samples = 8
