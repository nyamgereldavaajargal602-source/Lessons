# E-Mongolia Clone

This workspace contains a static clone of the deployed Vercel login page from `https://caffe-house-chi.vercel.app/more`.

## Files
- `index.html` — login page clone
- `profile.html` — local profile page after login
- `styles.css` — page styling
- `app.js` — login form behavior and navigation
- `manifest.json` — progressive web app metadata
- `icon.png` — downloaded app icon
- `favicon.ico` — downloaded favicon

## Run locally
Open `index.html` in your browser.

## Notes
The original Vercel app is built with Expo/Supabase. This clone reproduces the login UI and flow using static HTML/CSS/JS.
